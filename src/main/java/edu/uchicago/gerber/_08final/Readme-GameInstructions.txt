This game is an airplane shooting game

There are 3 lifes initially.
Left click the Mouse to start the game.
Move the mouse to control your plane.
If mouse is out of bound of the game screen, the game will pause.
If the game is in the pause state, then click the mouse(within the bound of game screen) to continue game.
Collision with Enemy plane can result in life subtraction(-1).
Game is over if your own plane's life is 0
If game over, click the mouse again to start a new game.

I have commented out the main function in Game.java and add the main function
in MyGamePanel.java.
